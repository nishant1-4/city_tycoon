# City Tycoon 🏙️

A Python OOP project simulating a living city economy — buildings, citizens,
resources, and disasters that all interact and affect each other over time.

This project is being built **incrementally**, one OOP concept at a time,
as a way to practice and demonstrate core and advanced Python OOP concepts
in a real, evolving codebase rather than isolated snippets.

## Concept

You manage a city. Citizens live in houses, work in buildings, earn money,
and pay taxes. Buildings produce or consume resources, decay over time, and
need maintenance. The city government sets policy. A turn-based simulation
runs the city forward in time, and it either thrives or collapses based on
the decisions made and the way the underlying objects interact.

## Goals

This project is intentionally designed to exercise nearly every major
Python OOP concept, including:

- Classes, objects, and constructors
- Encapsulation
- Inheritance and method overriding
- Abstraction (ABCs)
- Polymorphism
- Composition, aggregation, and association
- Operator overloading and magic methods
- Class variables, static methods, and class methods
- Custom exceptions
- File handling and JSON serialization
- Basic design patterns (Factory, Singleton, Observer)
